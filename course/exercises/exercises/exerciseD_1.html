Exercise D 1

Exercise:

3D Plot

In this exercise, we will repeat to plot the 3D data.

Data

The 3D data is given in the file grid_data.dat. The data contains the following columns: - \(r_\mathrm{OH_1}\) distance from O to H1 atom in Angstrom - \(r_\mathrm{OH_2}\) distance from O to H2 atom in Angstrom - \(V\) potential energy in kcal/mol

Data Path:

https://raw.githubusercontent.com/stkroe/PythonForChemists/main/course/data/exercises/3D_data/

Task

  • Load the data from the files grid_data.dat.
  • Create a 3D plot of the data using the plot_surface function.
  • Use a meaningful color map.
